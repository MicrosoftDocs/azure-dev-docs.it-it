# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Rete per la distribuzione di contenuti nella rete per la distribuzione dei contenuti di Azure

La [rete per la distribuzione di contenuti di Azure](https://azure.microsoft.com/services/cdn/) offre agli sviluppatori una soluzione globale per distribuire rapidamente agli utenti contenuti con esigenze di larghezza di banda elevata, come video, immagini o file statici, memorizzandoli nella cache in centinaia di punti di presenza in tutto il mondo. La rete per la distribuzione di contenuti può anche accelerare i contenuti dinamici, che non possono essere memorizzati nella cache, per ridurre i tempi di caricamento, risparmiare larghezza di banda e migliorare la velocità di risposta. Questi vantaggi si ottengono sia che si sviluppino o si gestiscano siti Web o applicazioni per dispositivi mobili oppure che si codifichino e distribuiscano contenuti multimediali in streaming, software di videogiochi, aggiornamenti del firmware o endpoint IoT.

## <a name="azure-content-delivery-network-features"></a>Funzionalità della rete per la distribuzione di contenuti di Azure

- Usare senza problemi servizi di Azure come applicazioni Web, servizi multimediali, archiviazione e servizi cloud.
- Gestire con facilità i picchi del traffico e i carichi pesanti.
- Integrare Akamai e Verizon per creare e personalizzare una strategia di distribuzione di contenuti.
- Usare l'accelerazione di siti dinamici.
- Controllare il comportamento di memorizzazione nella cache della rete per la distribuzione di contenuti con apposite regole.
- Ottenere supporto per domini personalizzati HTTPS.
- Visualizzare i log di diagnostica di Azure per esaminare l'analisi di base ed esportarli in Archiviazione BLOB di Azure, Hub eventi di Azure o Monitoraggio di Azure.
- Migliorare le prestazioni tramite compressione dei file.
- Usare filtri geografici per limitare i contenuti della rete per la distribuzione di contenuti in base a paese o area.
